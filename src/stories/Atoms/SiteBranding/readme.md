# Site Branding

Site branding takes a logo a label and a sub-label to form the site brand identifier for the header and footer.

<details>
  <summary>Twig Variables:</summary>
  ```
  logo: logo,
  label: "California Courts",
  sub_label: "Language Access",
  ```
</details>
