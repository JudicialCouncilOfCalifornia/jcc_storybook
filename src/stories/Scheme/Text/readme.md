# Example body/wysiwyg text

This page is to show how default "body/wyiswyg" content will appear with the base
elements styles.

This page includes:

- headings
- paragraphs
- links
- bold/strong
- italic/empasized
- ordered lists
- unordered list
- definition list
- code and preformatted

_This is only an example page and not to be used as a component_
