# Pills

This is a container for pills.
It creates a list of pill components, with optional icon.

<details>
  <summary>Twig Variables:</summary>
  ```
  variant: "default",
  icon_data: {
    icon: "translation",
    color: false,
  },
  items: [
    {
      variant: "default",
      label: "Pill",
      url: "#",
    },
    ...
  ],
  ```
</details>
