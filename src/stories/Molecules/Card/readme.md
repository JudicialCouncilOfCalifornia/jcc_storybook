# Card

Card components consist of a few sub components that can be displayed in multiple layouts.


<details>
  <summary>Inherited CSS Variables:</summary>  - `--color`
  - `--heading-color`
  - `--button-border`
  - `--button-bg`
  - `--button-fg`
</details>

<details>

  ```
  icon_data: {
    icon: "speaker",

  },
  heading: "Heading For A Card",
  text: "<p>...</p>",
  button: {
    label: "Button",
    href: "#",
    variant: "primary",
  },
  ```
